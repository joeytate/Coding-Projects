Basic computation functions created for assignments in Engineering Computations at UT Austin:

Bisect solves for the roots of an equation using the Bisection Method. 

FixedPoint solves for the roots of an equation using the Fixed Point Iteration method

GaussPivot solves a linear system of equations by pivoting the matrix and performing Gaussian Elimination

LUfactor outputs the the upper and lower matrices that result from the LU Factorization of the linear system

NewtRaph solves the roots of an equation using the Newton-Raphson Method

Powereig calculates the maximum eigenvalue for a system using the Power Method
